# APP CLONE NEXT

## 使用技術

- Next.js
- TypeScript
- Prisma

## Getting started (for 自分)

### Planet Scale のDBと接続

```sh
$ pscale connect my_database main --port 3309
# Secure connection to database my_database and branch main is established!
```

### ローカル環境で開発

```sh
yarn dev
# next
# started server on port 3000
```

Then, access <http://localhost:3000>
